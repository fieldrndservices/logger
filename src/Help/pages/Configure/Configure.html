Configure VIs

Owning Palette: Logger

Requires: Logger

Use the Configure VIs to configure the logger.

Palette Object Description
Configure Calendar Folder Tree Change the configuration for on-disk location of log files.
Configure File Change the configuration for the file logger.
Configure File Extension Change the file extension of log files.
Configure Level Change the maximum level for log statements.
Configure Maximum File Count Change the maximum number of files to have on-disk when a rollover occurs.
Configure Maximum File Size Change the maximum file size before creating a new log file.
Configure Prune All Change the pruning of all older files greater than the maximum file count.
Configure Root Folder Change the base, or root, location for new log files.
Configure User-Defined Codes Add user-defined error codes.
Configure User-Defined Descriptions Add user-defined descriptions for errors.
Configure Verbosity Change which errors are displayed to the user.
