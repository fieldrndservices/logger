Configure User-Defined Codes VI

Owning Palette: Configure VIs

Requires: Logger

Adds user-defined error codes to the logging statement so user-defined errors are properly formatted into Comma-Separated Values (CSV) lines.

Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.
User-Defined Codes is a list of error codes defined by a user.
Error Out contains error information. This output provides standard error out functionality.
