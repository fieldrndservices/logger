Configure Delimiter VI

Owning Palette: Configure VIs

Requires: Logger

Changes the default delimiter for columns, or fields, in each line, or record, of the log file. The default is the comma.

Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.
Delimiter is the character used to separate columns, or fields, in a line, or record within the log file. The default is a comma.
Error Out contains error information. This output provides standard error out functionality.
