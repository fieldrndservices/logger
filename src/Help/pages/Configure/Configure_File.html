Configure File VI

Owning Palette: Configure VIs

Requires: Logger

All of the configurations for the default log file can be configured with this one VI instead of using VIs for each individual configuration. The individual configuration VIs are just wrappers around this VI.

Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.
Configuration is all of the configurations for the default log file.
Error Out contains error information. This output provides standard error out functionality.
