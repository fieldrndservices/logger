Create Temporary Root Folder VI

Owning Palette: Utility VIs

Requires: Logger

Creates a folder tree for the default log file in the user's temporary directory.

Folder Names is a list of folders with the each element in the list created, or nested, in the previous element's folder.
Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.
Temp Log Folder is the root folder in the user's temporarily directory.
Error Out contains error information. This output provides standard error out functionality.
