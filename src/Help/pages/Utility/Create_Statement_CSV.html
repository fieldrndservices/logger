Create Statement CSV

Converts a logging statement to a Comma Separated Values (CSV) line.

Statement is a logging statement.

    Time Stamp is the time and date the logging statement is created. This is used to create an ISO 8601 timestamp for the first column of the CSV line, or row.

    Tag is the related level, or tag, for this statement.

    Source is the VI or location within the application that created the statement.

    Message is the user-defined message to include as the last column in the CSV line, or row.

Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.

Error Out contains error information. This output provides standard error out functionality.

CSV is the Comma-Separated Values (CSV) line of text.
