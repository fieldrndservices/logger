Utility VIs

Owning Palette: Logger

Requires: Logger

Use the Utility VIs to handle errors.

Palette Object Description
Catch Error Handle and log an error.
Create Statement CSV Create a CSV line of text from a log statement.
Create Temporary Root Folder Create a location for log files in the user's temporary folder.
Disable Default File Disable the default file logger.
Mask Errors Log all errors but clear any from a list of error codes.
