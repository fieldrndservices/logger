VI and Control Reference

Use the VIs and controls to write log statements to listeners.

Palette Object Description
Initialize Initializes the logger.
Shutdown Shuts down the logger.
Action-Status VIs Use these VIs to start and stop logging, add log statements, and register listeners.
Configure VIs Use these VIs to configure the logger.
Data VIs Use these VIs to read data from the logger.
Utility VIs Use these VIs to handle errors.
Controls Controls and type definitions used within the library.
