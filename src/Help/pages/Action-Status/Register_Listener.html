Register Listener VI

Owning Palette: Action-Status VIs

Requires: Logger

Creates a new listener for logging statements. By default, there is at least one listener, which is the internal listener used for the default log file.

Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.
Listener ID is an unique string to identify the listener.
Error Out contains error information. This output provides standard error out functionality.
