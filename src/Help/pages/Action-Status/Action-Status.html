Action-Status VIs

Owning Palette: Logger

Requires: Logger

Use the Action-Status VIs to start and stop logging, add log statements, and register listeners.

Palette Object Description
Debug Log a DEBUG statement.
Error Log an ERROR statement.
Fatal Log a FATAL statement.
File Run Run the file logger.
File Start Start the file logger.
File Stop Stops the file logger.
Info Log an INFO statement.
Register Listener Add a listener for log statements.
Trace Log a TRACE statement.
Unregister Listener Remove a listener of log statements.
Warn Log a WARN statement
