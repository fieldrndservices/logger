Configure Level

Sets the global level for all listeners.

The higher the value, the more logging statements that are sent to listeners. Generally, a level of zero (0) disables all logging statements and a level of seven (7) enables all logging statements. Any logging statement with a level less than or equal to the global level is sent to all listeners. Logging statements are tagged as one of the following: (1) FATAL, (2) ERROR, (3) WARN, (4) INFO, (5) DEBUG, and (6) TRACE. So, if a level of 4 is used, then all logging statements tagged as FATAL, ERROR, WARN, or INFO are sent to listeners, where one of the listeners is the default log file.

Level determines which logging statements to write to the default log file and other registered listeners based on the logging statement's tag (FATAL, ERROR, WARN, INFO, DEBUG, and TRACE). A zero (0) disables all logging. A value of seven (7) or higher will write and broadcast all logging statements.

Error In describes error conditions that occur before this node runs. This input provides standard error in functionality.

Error Out contains error information. This output provides standard error out functionality.
